Общая информация о проекте

Сайт был создан в апреле 2025 года в рамках учебного проекта. Разработан данный сайт с использованием Django. Сайт представляет собой сборник популярных итальянских мемов и звуков к ним. Создан с особой любовью и внимательностью к деталям! :3

Сайт Memes имеет 3 страницы: "Главная", "О разработке и "Сотрудничество"

Технологии и инструменты

Этот проект был создан с использованием различных технологий и инструментов. В процессе разработки использовались следующие компоненты:

  • Python 3.9
  • Django 4.2
  • HTML5 + CSS3
  • SQLite для хранения данных
  • Sublime text / PyCharm для написания кода
